# How to find your installed version #

  1. Open PowerShell
  1. Type `$Host.Version`

For example:

## PowerShell 1.0 ##

```
PS H:\> $Host.Version

Major  Minor  Build  Revision
-----  -----  -----  --------
1      0      0      0
```

## PowerShell 2.0 ##

```
PS H:\> $Host.Version

Major  Minor  Build  Revision
-----  -----  -----  --------
2      0      -1      -1
```
